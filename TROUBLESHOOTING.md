# Решение проблем

## Как скопировать ошибки из Expo

1. **В терминале:**
   - Ошибки отображаются в консоли, где запущен `npm start`
   - Скопируйте весь текст ошибки

2. **В приложении Expo Go:**
   - Нажмите на красный экран с ошибкой
   - Прокрутите вниз, чтобы увидеть полный стек ошибок
   - Сделайте скриншот или перепишите текст

3. **В браузере (если используете Expo Web):**
   - Откройте DevTools (F12)
   - Перейдите на вкладку Console
   - Скопируйте ошибки

## Частые проблемы

### 1. Ошибка с react-native-maps
Если видите ошибку связанную с картами:
- Карта временно отключена в MapScreen для диагностики
- После исправления других ошибок можно включить обратно

### 2. Ошибка с навигацией
Убедитесь, что установлены все зависимости:
```bash
npm install
```

### 3. Ошибка с Supabase
Проверьте, что:
- Интернет подключен
- Supabase URL и ключ правильные в `src/lib/supabase.ts`

### 4. Ошибка с модулями
Попробуйте:
```bash
npm install
npm start -- --clear
```

## Отправка ошибок

При отправке ошибки укажите:
1. Полный текст ошибки
2. На каком экране она появилась
3. Что вы делали перед ошибкой
